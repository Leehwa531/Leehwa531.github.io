# 백엔드 맛집, 박가네 (Park's Backend Restaurant)

이 프로젝트는 백엔드 개발자 포트폴리오 사이트입니다.

## 🚀 실행 방법 (How to Run)

로컬 환경에서 사이트를 확인하는 방법입니다.

### 방법 1: VS Code Live Server (추천)
VS Code를 사용 중이라면 'Live Server' 확장 프로그램을 설치하고 우측 하단의 `Go Live` 버튼을 클릭하세요.

### 방법 2: Python 사용
터미널에서 다음 명령어를 실행하세요:
```bash
python -m http.server
```
그 후 브라우저에서 `http://localhost:8000`으로 접속하세요.

### 방법 3: Node.js (npx) 사용
터미널에서 다음 명령어를 실행하세요:
```bash
npx serve
```
그 후 브라우저에서 `http://localhost:3000`으로 접속하세요.

### 방법 4: 직접 열기
`index.html` 파일을 더블 클릭해서 브라우저로 엽니다. (일부 기능이 제한될 수 있음)
